### Set of tiny terminal tools that will be useful if you would ever want to use linux without a DE/WM

**List of all tools/scripts that are included:**
- Calculator
- Clock
- Notes App
- Weather App
- Password Generator
- Currency Converter
- Unit Converter (lb -> kg, ft -> m, etc.)
- Random Quote Generator
- Guessing Game

**Requirements:**
- git
- gcc
- node
